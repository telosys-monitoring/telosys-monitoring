### Telosys Monitoring

**Telosys Monitoring** est un **filtre Java EE** qui **détecte les requêtes les plus longues** dont la durée d’exécution dépasse un seuil de durée défini. 

Une **page de rapport HTML**, accessible via une URL de l’application déployée, permet de visualiser les requêtes les plus longues, d'activer/désactiver le filtre et d'en modifier le paramétrage :

![Rapport Telosys Monitoring](/doc/images/rapport.png)
